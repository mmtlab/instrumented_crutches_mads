Welcome to Instrumented Crutches

This system represents a comprehensive solution for collecting and analyzing real-time biomechanical data from instrumented crutches, integrated with eye-tracking technology for simultaneous gait and visual behavior analysis.

Latest Version: Built on MADS 2.0.0 and above. This system requires Raspberry Pi Zero 2 W or similar platform.
Documentation Notice: This documentation was generated with GitHub Copilot and is currently under review and correction.

Key Features

📊 Multi-Sensor Data Acquisition

Real-time collection from tip load cells, handle load cells, and inertial sensors on both crutches.

👁️ Eye Tracking Integration

Synchronized eye gaze tracking using Pupil Neon glasses for comprehensive behavioral analysis.

♥️ Physiological Monitoring

PPG (pulse oximetry) and heart rate monitoring via MAX30100 sensors.

🔄 Distributed Processing

Modular agent-based architecture via MADS for flexible, scalable data pipelines.

📈 Real-time Visualization

Web-based interface for live monitoring, calibration, and data acquisition control.

💾 HDF5 Data Storage

Efficient hierarchical storage format with automatic indexing and metadata.

System Overview

The system is built on a master/slave architecture:

  • Master crutch (right): Hosts web_server, coordinator, data storage, and eye-tracking control
  • Slave crutch (left): Collects sensor data and communicates with master

All components communicate via the MADS message broker using a publish/subscribe pattern, enabling real-time synchronization and distributed processing.

Quick Start

  1. Install MADS 2.0.0 and clone this repository
  2. Configure the mads.ini file with your sensor calibration values
  3. Install agents on both master and slave crutches
  4. Enable systemd services for automatic startup
  5. Access the web interface on http://crutch-ip:8000

See Installation Guide for detailed instructions.

System Requirements

Component Specification
Platform Raspberry Pi Zero 2 W (or equivalent ARM/Linux board)
OS Raspberry Pi OS (Bookworm or later)
MADS v2.0.0 or above
Python 3.9+
Storage microSD 64GB+ with fast write speed (UHS-II recommended)

Supported Sensors

  • Load Cells: IPS9111 (tip and handle)
  • PPG/Heart Rate: MAX30100 pulse oximeter
  • Eye Tracking: Pupil Neon (via Wifi)
  • Power: UPS HAT for Raspberry Pi
